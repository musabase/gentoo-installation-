# XFCE on Gentoo Linux

This document provides structured notes for installing and running
the XFCE desktop environment on Gentoo Linux.

XFCE is a lightweight, stable, and modular desktop environment that
works well with Gentoo’s source-based model.

---

## Why Choose XFCE on Gentoo

- Lightweight and resource-efficient
- Traditional desktop workflow
- Modular design
- Good balance between performance and usability

---

## What Is Required Before Installing XFCE

- A working Gentoo base system
- Correct profile selection (desktop profile recommended)
- Updated Portage tree
- Proper USE flags configuration
- Xorg and graphics drivers configured

---

## High-Level Installation Flow

1. Select appropriate desktop profile
2. Configure USE flags for graphical stack
3. Install Xorg server and drivers
4. Install XFCE meta package
5. Configure session startup
6. Launch XFCE session

---

## Notes

- Compilation time depends on hardware
- More CPU cores reduce build time
- Proper USE flag configuration avoids rebuild issues
- Always refer to the Gentoo Handbook when needed
